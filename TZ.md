Менеджер веб-автоматизации на базе CloakBrowser ## Язык
Python 3.10+ ## Суть
Оркестратор, который поднимает внешний pip-пакет `cloakbrowser` и гоняет веб-автоматизацию по нескольким аккаунтам. Никаких браузерных расширений не используем, кошелёк инъектим через JS в страницу.

## Файл accounts.json
Генерится автоматически скриптом `create_family.py`. В поле `email` оставляем пустым, остальное заполняем.

Имена аккаунтов: `sibil_1`, `sibil_2` и т.д. Структура:
```json
[ { "name": "sibil_1", "email": "", "evm": { "address": "0x...", "private_key": "0x..." } }
]
``` ## Скрипт генерации create_family.py
- Генерит N аккаунтов (имена `sibil_1`, `sibil_2`...).

- Для каждого генерит случайную BIP39 сид-фразу. - Из сида выводит приватный ключ и адрес.

- Адрес + ключ пишет в `accounts.json` (email пустой). - Сид-фразу + имя аккаунта пишет в отдельный `seeds.json` (секретный файл, юзер его прячет).

## EVM-инъекция в браузер
Через `page.evaluate()` внедряем в `window` объект провайдера (аналог `window.ethereum`) с приватным ключом из конфига, чтобы сайты думали, что стоит MetaMask. Без расширений.

## Логин на сайте
На сайте логин требует подписи `SIGN` через кошелёк. Скрипт инициирует подпись сообщения `SIGN` инъектированным провайдером и прокидывает результат в форму/запрос для входа.
